js加载的缺点:加载工具方法没必要阻塞文档， 过得js加载会影响页面效率，一旦网速不好， 那么整个网站将等待js加载而不运行后续渲染等工作；
有些工具方法按需加载,用到再加载， 不用不加载。

这个javascript从此就变成了异步加载了，但是只有IE9以下浏览器可以使用----->defer是IE给自己提供的方法IE9以下可以顺利的使用 ，defer标签的执行时刻，不会下载完就立刻执行，而是整个页面解析完毕时，才会执行
<script  src="路径/标签内部写js内容"   type="text/javascript"  defer ></script>

async是W3C标准方法 async异步加载，和defer不同，async是加载完立刻执行，执行也是异步执行 非IE浏览器使用，IE高版本也可以使用，如:IE9以上，符合W3C标准就可以使用 浏览器可以使用----->但是async只能执行外部脚本 如：src="路径"不能在标签内，写js内容了 <script src="路径"   type="text/javascript"   async ></script>
