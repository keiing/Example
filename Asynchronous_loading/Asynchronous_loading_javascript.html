JSON

先来谈谈json格式 什么是json格式 '{"name":"deng","age":123}' 这就是json格式 json字符串
JSON.parse()将json字符串装换为对象，称反序列化 JSON.stringify()将对象转换为json字符串,称序列化
再来谈谈什么是json JSON是一种传输数据的格式(以对象为样板，本质上就是对象， 但用途有区别，对象就是本地用的，json是用来的传输的) JSON.parse();string——> json
JSON.stringify();json——>string
